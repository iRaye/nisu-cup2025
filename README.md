# Parking Coupon Account Page

이 저장소는 GitHub Pages를 통해 호스팅할 수 있는 간단한 HTML 페이지입니다.
- **파일명**: index.html (자동으로 기본 페이지로 인식됩니다)
- **목적**: 방문자가 스마트폰이나 브라우저에서 계좌번호를 한 번에 복사할 수 있도록 합니다.

## 🚀 배포 방법
1. GitHub에 새 저장소를 생성합니다 (예: `parking-coupon`).
2. 이 폴더(`index.html`, `README.md`)를 업로드합니다.
3. GitHub 저장소 → Settings → Pages → Source: `Deploy from a branch` → Branch: `main` / `(root)` 설정 후 Save.
4. 약 1분 후 `https://username.github.io/parking-coupon/` 주소에서 페이지를 열 수 있습니다.
